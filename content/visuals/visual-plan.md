# Visual Plan — Premium Management Consulting Website

## Design Principles
- **Minimal premium style**: Wide spacing, clean typography, subtle borders, rounded corners
- **Mobile-first, responsive**
- **Lucide-react icons only** (no external images yet; use placeholders for hero abstracts)

## Page-by-Page Visual Mapping

### Home (/)
- Hero: placeholder (abstract gradient/geometric)
- 4 icon cards (services/offerings)
- 5-step stepper (our approach)
- CTABox

### Services (/services)
- 4 icon cards (service categories)
- Framework diagram: simple 2x2 block layout
- CTABox

### Service Pages (/services/[slug])
Examples: strategy, operations, transformation, people
- Mini hero placeholder
- 3 pain-point icon bullets
- Deliverables list (with check icons)
- 5-step stepper
- CTABox

### Methodology (/methodology)
- 5-step stepper (primary)
- Toolbox icons (methods/tools)
- CTABox

### Cases (/cases)
- Icon for each case category
- Card layout with icons
- CTABox

### Resources (/resources)
- Icon per resource type (article, whitepaper, video)
- List/card layout
- CTABox

### About (/about)
- Team/values icons
- Clean sections with icon accents
- CTABox

### Contact (/contact)
- Contact form icons (mail, phone, map)
- Form + info layout
- CTABox

## Icon Usage (lucide-react)
- Home: Target, Zap, BarChart3, Users
- Services: Briefcase, Layout, TrendingUp, Building2
- Pain points: AlertCircle, Clock, PieChart
- Deliverables: CheckCircle2
- Stepper: Circle (numbered 1–5)
- CTA: ArrowRight, MessageCircle
