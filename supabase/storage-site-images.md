# Resim yüklemek için Storage bucket (bir kez)

Panelden "Resim seç" ile yüklenen dosyalar Supabase Storage'da tutulur.

1. Supabase Dashboard → **Storage**
2. **New bucket** → İsim: `site-images`
3. **Public bucket** işaretli olsun (herkes görüntüleyebilsin)
4. Create

Bundan sonra paneldeki "Resim seç" ile yüklediğiniz resimler bu bucket'a gider.
